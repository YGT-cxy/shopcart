# shopping-mall
```
vue开发的一个购物车Demo页面项目，包括商品数量的增减和导航列表的联动及计算金额等功能。
```

## 安装项目需要的依赖包
```
npm install
```

### 启动一个本地web服务器预览项目，端口号为8080
```
npm run serve
```

### 打包静态资源
```
npm run build
```
